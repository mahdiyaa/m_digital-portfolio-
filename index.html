UX Design Project 2025

Overview: A conceptual website designed for an NYP Debate CCA to clearly communicate activities, events, and achievements to students.

My role: I planned the layout and user flow, designed key pages, and focused on improving readability and navigation so users can quickly find information such as training schedules, event updates, and sign-up details.

Key UX focus: Clean information hierarchy, consistent visual style, and a student-friendly design that feels modern and easy to use.

Outcome: A structured, realistic prototype that shows how a CCA could present content professionally to attract new members and keep current members informed.

Web Development Project 2026

Overview: A community web application built as a shared hub for youth and the elderly to connect, participate, and support each other through shared stories, activities, and updates.

My role: I developed the web app structure and UI, designed pages for community interaction, and ensured the layout is simple, accessible, and easy for users of different ages.

Tech stack: HTML, CSS, Python, and Flask (with a modular structure that allows future features like login, posting, and topic categories).

Why it matters: The platform is designed to reduce social gaps by making it easier for different generations to communicate and share resources in one place.

Outcome: A working web application foundation that is scalable and ready to expand with more community features.
